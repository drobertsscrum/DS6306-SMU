DS6306 SMU Data Science Portfolio

This repository contains my coursework, case studies, and applied projects from DS6306 (Doing Data Science) at Southern Methodist University.

My focus throughout this work is not just on building models, but on understanding the data, making thoughtful decisions, and communicating insights clearly.

What you will find here
- Structured data workflows from raw to processed datasets
- Statistical modeling and machine learning in R
- Visualizations designed to explain, not just display results
- Case studies that connect analysis to real world decisions

Repository Structure
 data: Raw and processed datasets
 scripts: Data preparation, modeling, and visualization code
 notebooks: R Markdown analysis and exploratory work
 outputs: Figures, tables, and final results
 presentations: Slides and executive summaries
 docs: Supporting documentation and notes

Tools and Technologies
 R and RStudio
 tidyverse, ggplot2, caret, e1071
 shinyapps
 R Markdown for reproducible analysis

Approach

I approach data science as both a technical and strategic discipline.
